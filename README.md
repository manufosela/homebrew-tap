# Homebrew Tap for Karajan Code

Local multi-agent coding orchestrator with TDD, SonarQube, and automated code review.

## Install

```bash
brew tap manufosela/tap
brew install karajan-code
```

## Usage

```bash
kj --version
kj init
kj run "Create a utility function with tests"
kj-tail              # Monitor pipeline in another terminal
```
